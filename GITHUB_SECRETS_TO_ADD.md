## How to Add Secrets

1. **Go to the GitHub Secrets page:**
   https://github.com/SwipeSavdev/camp-card/settings/secrets/actions

2. **For each secret above:**
   - Click "New repository secret"
   - Enter the Name (EC2_HOST, EC2_USER, or EC2_SSH_PRIVATE_KEY)
   - Copy and paste the Value exactly as shown above
   - Click "Add secret"

3. **Verify all three secrets are added**
   You should see:
   - EC2_HOST
   - EC2_USER
   - EC2_SSH_PRIVATE_KEY
