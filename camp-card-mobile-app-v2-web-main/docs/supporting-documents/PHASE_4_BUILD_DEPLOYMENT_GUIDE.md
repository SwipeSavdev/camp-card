## TROUBLESHOOTING GUIDE

### If App Doesn't Start
```
1. Clear cache:
 npm run clean:node
 npm install

2. Rebuild native code:
 rm -rf ios android
 npx expo prebuild

3. Check logs:
 Check Metro bundler output
 Check device system logs
```

### If Compilation Fails
```
1. Run type check:
 npm run type-check

2. Check imports:
 npm run lint

3. Verify dependencies:
 npm ls [package-name]
```

### If API Calls Fail
```
1. Verify backend is running:
 curl http://localhost:8080/actuator/health

2. Check network connectivity:
 Use device network inspector

3. Verify API endpoints:
 Check backend documentation
```

### If UI Looks Wrong
```
1. Clear device cache:
 Restart simulator/device

2. Verify theme colors:
 Check src/uiux/theme.ts

3. Check screen size:
 Test on different devices
```
